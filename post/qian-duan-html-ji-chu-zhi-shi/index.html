前端-HTML基础知识

· 2019-07-31 ·

在前端中，HTML，CSS 和 JavaScript 各有自己的作用。打个比方，假如前端是一个人，那么 HTML 属于骨骼，支撑起整个结构，CSS 属于衣服，管理样式，而 JavaScript 属于肌肉，动态调整页面的样式和内容。

在开头我们先介绍一个网站，关于 Web 开发的所有资料都可以在这个网站找到
Mozilla

标签


<p>Hello World !!</p>

// <p> 为开始标签，</p> 为结束标签

比如代码上面 的 p 就是标签，标签中间的为内容。

格式为：元素 = <开始标签>内容<结束标签>

空元素

如果没有内容的话，则这个元素为空元素，比如图片标签和换行


<img src="xxxxxxx" />

<br /> // 换行

属性

可以在标签里面增加属性，比如图片里面的 src 就是属性，它的值是图片的链接。


// class = "property"  这个就是属性
<p class = "property">Hello World !!</p>

Hello World

刚开始学习一些语法的时候，如果还不行用一些代码编辑器或者 IDE，我们可以找一些线上的工具练习使用。比如下面的两个网站，可以在线编辑代码，实时浏览效果，但是代码提示就没有那么友好了。

中文：
http://code.ftqq.com

英文：
https://codepan.net


// 说明文档类型
<!doctype html>

<html>
  <head>
  	<meta charset = "UTF-8"  />    // 字符集
		<meta name="keywords" content ="课程" /> // 关键词
		<meta http-equiv="refresh" content ="30" />  // 定时刷新
		<meta name="viewport" content ="width=device-width, initial-scale=1.0" />
		
    <title>Hello World Demo !!</title>  // 显示在浏览器的 tab 栏
  </head>
  <body>
    <p>
      Hello World !!
    </p>
  </body>
</html>

第一个简单的 Hello World 的例子。

  1. 字符集：charset 是网页支持的元数据字符集，一般我们就写 UTF8 的。
  2. 关键词：name content 为关键词，有助于搜索引擎的优化
  3. 定时刷新
  4. 指定 ViewPort (让网页的内容更加适合设备的大小)
  • ASCII 128 个字符，主要以英文为主：http://www.asciitable.com/
  • GB2313 收录了6763个汉字
  • UTF8 每个字符包含1~6个字节
    1. US-ASCII: 1个字节
    2. 带有附加符号的拉丁文、希腊文等：2个字节
    3. 汉字：3个字节
    4. 其他：4~6个字节

head 中的其他元素

  • style：页面内的 CSS
  • link：引入独立文件中的 CSS
  • script：引入 JavaScript 文件
  • base：指定页面中链接的相对路径

base 里面指定的是 页面中 a 标签跳转的页面


<base target="_blank">  //  让网页中所有 a 标签的点击都新开一个标签页

body 里面的元素

body 就是网页的主页面

文字样式

1.h1~h6 标题


<h1>一级标题</h1>
<h2>二级标题</h2>
<h3>三级标题</h3>
<h4>四级标题</h4>
<h5>五级标题</h5>
<h6>六级标题</h6>

2.加粗

<p>
      <strong>我的博客</strong>
</p>
<p>
      <b>我的博客</b>
</p>

斜体

英文里面作强调的时候经常使用斜体


<em>我的博客</em>

small

可以作标注使用，让用户注意力放在正文上面。


<small>我的博客</small>

ins（下划线）和 del（删除）


<del>我的博客</del>
<ins>我的博客</ins>

<span class="underline">我的博客</span>

上下标和 abbr


<p>
      X<sup>2</sup> &amp; X<sub>2</sub>
</p>

<p>
	什么是 <abbr title="Hypertext Markup Language">HTML</abbr>
</p>

上下标一般用来数学公式表示，abbr 是给一些缩写增加说明。如图，HTML 是底部条虚线标识，当鼠标移动到上面，会有 HTML 的全称 Hypertext Markup Language 显示出来。

链接


<p>
      <!-- target="_blank" 点击链接时新开一个tab展示 -->
      我的博客是：<a href="https://zhengyanfeng.github.io" target="__blank">fat-panda</a>
    </p>
    
    <p>
      <!-- target="fatpanda标识" 点击链接时会重新加载打开对应网页 -->
      我的微博是：<a href="https://www.weibo.com/p/1005052309146301/home?from=page_100505&mod=TAB&is_all=1#place" target="fatpanda">fat-panda</a>

段落 p & br

br 为 换行符，p 标签已经默认给了一个段落之间行间距。

CSS 里面可以用 margin 更改行间距，line-height 更改行高，通过这些调整，使得文字在网页中显示更加美观。

段落 pre & code

pre 可以保留文字原来的格式
code 使中间的字体使用等宽字体

列表

ol 为有序列表
ul 为无序列表


<body>
    <h1>
      诗人列表
    </h1>
    <ol>
      <li>李白</li>
      <li>白居易</li>
      <li>李商隐</li>
    </ol>
    
    <h1>
      古诗列表
    </h1>
    <ul>
      <li>长恨歌</li>
      <li>琵琶行</li>
      <li>蜀道难</li>
    </ul>
    
  </body>

dl 子列表

dl 为外部标签，dt 和 dd 为内部的，dd 为 dt 的子列表
