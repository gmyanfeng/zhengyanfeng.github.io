Flutter-Image 图片组件

· 2019-07-23 · # flutter

导入图片的方式有以下几种：

  • Image.asset : 加载项目资源目录中的图片，用的是相对路径，导入后会增加包的体积。
  • Image.network : 网络图片资源
  • Image.file : 加载本地图片，使用的是绝对路径，和包大小无关
  • Image.memory : 加载 Uint8List 资源的图片
import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Flutter Demo',

      home: Scaffold(
        body: Center(
          child: Container(
            child: new Image.network(
              'https://flutter-io.cn/favicon.ico',
              scale: 1.0,
            ),

            width: 300,
            height: 200,
            color: Colors.lightBlue,
          ), //child
        ), //body
      ), //home
    ); //MaterialApp
  }

fit 属性

fit 属性控制图片的拉伸和挤压

  • BoxFit.fill : 全图显示，图片会被拉伸，充满父容器
  • BoxFit.contain : 显示原比例
  • BoxFit.cover : 显示可能被拉伸，裁切，充满整个容器而且不会变形
  • BoxFit.fitWidth : 宽度充满，显示可能拉伸，可能裁切
  • BoxFit.fitHeight : 高度充满，显示可能拉伸，可能裁切
  • BoxFit.scaleDown : 效果和 contain 相似，但是不允许显示超过图片的大小

图片的混合模式

图片混合模式（colorBlendMode）和 color 属性配合使用，能让图片改变颜色。

child: new Image.network(
              'https://flutter-io.cn/favicon.ico',
              scale: 1.0,
              color: Colors.greenAccent,
              colorBlendMode: BlendMode.colorBurn,
            ),

repeat 图片重复

  • ImageRepeat.repeat : 横向和纵向都进行重复，直到铺满
  • ImageRepeat.repeatX : 横向铺满
  • ImageRepeat.repeatY : 纵向铺满
child: Container(
            child: new Image.network(
              'https://flutter-io.cn/favicon.ico',
              scale: 1.0,
              color: Colors.greenAccent,
              colorBlendMode: BlendMode.colorBurn,

              repeat: ImageRepeat.repeat,
            ),

            width: 600,
            height: 900,
            color: Colors.lightBlue,
          ), //child
