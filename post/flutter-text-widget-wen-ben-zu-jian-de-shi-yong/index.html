Flutter-Text Widget 文本组件的使用

· 2019-07-23 · # flutter

在 Flutter 里面一切皆组件（Widget）。

上篇文章我们运行了一个简单的 Hello World 程序，里面已经包含了一个文本组件，不过我们用的都是默认的属性。

Text Widget

我们重新创建一个新的项目，删除默认的模板代码，实现一个 Text Widget，代码如下：

import 'package:flutter/material.dart';

//主函数
void main() => runApp(MyApp());
//声明 MyApp 类
class MyApp extends StatelessWidget{
  // 重写 build 方法
  @override
  Widget build(BuildContext context){
    //返回一个Material 风格组件
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: Scaffold(
        appBar: AppBar(
          //app bar 的 title， 可以类比于 iOS 的 navigationbar 的 title
          title: Text('Text Widget 文本组件'),
        ),
        // 添加一个 Hello World 的文本
        body: Center(
          child: Text('Hello World，这是第一个程序，这里是 zhengyanfeng.github.io 博客'),
        ),
      ),
    );
  }
}

TextAlign 文本对齐方式

  • center：居中对齐
  • left：左对齐
  • right：右对齐
  • start：以开始位置对齐
  • end：以文本结尾对齐
// 添加一个 Hello World 的文本
        body: Center(
          child: Text(
              'Hello World，这是第一个程序，这里是 zhengyanfeng.github.io 博客',
               textAlign: TextAlign.center,
          ),
        ),


其他的效果可以自己修改看看。

maxLines 属性

设置显示的行数，如果我们设置为1，就是只能单行显示

// 添加一个 Hello World 的文本
        body: Center(
          child: Text(
              'Hello World，这是第一个程序，这里是 zhengyanfeng.github.io 博客',
               textAlign: TextAlign.center,
               maxLines: 1,
          ),
        ),

overflow 属性

当文字超出文本框时，如何处理？

  • clip：切断，省略剩下的文字
  • ellipsis：显示省略号
  • fade：有一个渐变消失的效果
// 添加一个 Hello World 的文本
        body: Center(
          child: Text(
              'Hello World，这是第一个程序，这里是 zhengyanfeng.github.io 博客',
               textAlign: TextAlign.center,
               maxLines: 1,
               overflow: TextOverflow.fade,
          ),
        ),

style 属性

这个属性比较多，具体可以查询文档。下面实现字体大小为25.0，颜色为粉色，并且有个下划线。

// 添加一个 Hello World 的文本
        body: Center(
          child: Text(
              'Hello World，这是第一个程序，这里是 zhengyanfeng.github.io 博客',
               textAlign: TextAlign.center,
               maxLines: 1,
               overflow: TextOverflow.ellipsis,

               style: TextStyle(
                 fontSize: 25.0,
                 color: Color.fromARGB(255, 255, 150, 150),
                 decoration: TextDecoration.underline,
                 decorationStyle: TextDecorationStyle.solid,
               ),
          ),
        ),
