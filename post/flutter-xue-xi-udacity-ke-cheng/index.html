Flutter学习-HelloWorld

· 2019-07-18 · # flutter

什么是Flutter？这是一个Google出的一个跨平台的框架，支持Android和iOS。大家可以去搜索下，这个文章系列是记录我跟着Udacity课程的学习过程。

  1. Flutter 的官网
  2. Flutter 中文网站
  3. Udacity Flutter课程

环境搭建

开发环境的搭建，可以去 Flutter 官网看看步骤，介绍的很详细。我自己选择的是macOS 平台，IDE 用的是 Android Studio (因为可以直接创建 Flutter 的工程)。

Flutter 官网搭建教程：https://flutter.io/setup
IDE 选择：https://flutter.io/using-ide.

相关的参考链接：

Hello World

打开 Android Studio，选择 Start a new Flutter project

这个选择是需要安装插件才有的，在 Android Studio 的菜单栏中选择 Preferences -> Plugins，然后在搜索框中搜索 Flutter，安装的过程中还会提示你是否安装 Dart (Flutter 的开发语言)，默认同意继续就好了。

出现配置应用的窗口：

  • Project name: flutter_app_helloworld
  • Flutter SDK path: 自己选择之前配置的 Flutter 开发环境目录（比如我的是放在了这个文件路劲下：~/Documents/development/flutter）
  • Project Location：自己选择项目存放的位置了
  • Description：随便自己添加描述
  • Create project offline -> 这个自己选择，个人建议可以勾选。

然后可以选择 Next，接着出现下一个页面：

  • Company domain: 公司的域名反向（比如 com.baidu）
  • 勾选上 Include Kotlin support for Android code (支持安卓)
  • 勾选上 Include Swift support for iOS code (支持 iOS)

点击 Finish 开始创建项目

运行项目

废话不多说，先跑起来看看，我选择运行在 iOS 的模拟器上。如图，选择 Open iOS Simulator，系统会启动一个模拟器（当然这步需要你之前已经安装了 Xcode，iOS 开发的 IDE），选择 iPhone XR 模拟器，然后点击Debug。等待模拟器启动并打开项目，默认的项目如下图所示。

Flutter 项目的代码主要在 main.dart 文件，打开该文件，把默认的代码全部删除。Hello World 的代码如下：

import 'package:flutter/material.dart';

//主函数
void main() => runApp(MyApp());

//声明 MyApp 类
class MyApp extends StatelessWidget{

  // 重写 build 方法
  @override
  Widget build(BuildContext context){
    //返回一个Material 风格组件
    return MaterialApp(
      title: 'Welcome to Flutter',
      home: Scaffold(
        appBar: AppBar(
          //app bar 的 title， 可以类比于 iOS 的 navigationbar 的 title
          title: Text('Hello World'),
        ),
        // 添加一个 Hello World 的文本
        body: Center(
          child: Text('Hello World'),
        ),
      ),
    );
  }
}
