macOS 中配置门罗币(XMR)挖矿

· 2020-01-11 ·

门罗币(Monero) 是强调隐私和安全的数字货币，具体的信息我就不介绍了，大家可以去官网了解。

官网地址：https://web.getmonero.org/zh-cn/index.html

维基百科：门罗币介绍

CoinMarketCap 币价：￥334.87（2019.12.20）

废话不多说，我们进入配置步骤。首先我们来到挖矿的网页：https://xmrig.com/

1.点击 Wizard

2.添加配置，点击 New configuration

点击 Add pool，然后选择 supportXMR.com 这个矿池地址，配置中输入你自己的钱包地址，如图

3.配置 Backends

点击选择 Backends，打开 CPU，其他的可以自己选择

4.选择捐献的比例

点击 Misc，去选择捐献给该网站的比例，默认为 5%

到此矿池已经配置好了，接下来用去下载源码，在 macOS 下编译

5.点击 Docs 查看文档，选择 xmrig -> build from source -> macOS

按照文档的步骤配置环境

6.安装 Homebrew

Homebrew 官网

打开终端，输入命令安装 Homebrew

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

7.安装依赖

brew install cmake libuv openssl hwloc

安装完毕后，下载源码

Github 下载源码

找个存放源码的文件夹，在终端的输入命令下载源码

git clone https://github.com/xmrig/xmrig.git

创建 build 文件夹

mkdir xmrig/build

在终端切到 xmrig 文件夹下

cd xmrig

下载 hwloc 文件夹

curl -O https://download.open-mpi.org/release/hwloc/v2.1/hwloc-2.1.0.tar.bz2

解压缩包

tar xjf hwloc-2.1.0.tar.bz2

切到 hwloc-2.1.0 文件夹

cd hwloc-2.1.0

配置 hwloc

./configure --disable-shared --enable-static --disable-io --disable-libxml2

安装下面的命令继续编译

make -j$(sysctl -n hw.logicalcpu)
cd ../build
cmake .. -DOPENSSL_ROOT_DIR=/usr/local/opt/openssl -DHWLOC_INCLUDE_DIR=../hwloc-2.1.0/include -DHWLOC_LIBRARY=../hwloc-2.1.0/hwloc/.libs/libhwloc.a
make -j$(sysctl -n hw.logicalcpu)

上述的步骤都顺利走完了的话，到此就完成了所有的配置

从终端进入到我们 build 后的文件夹，如下图

然后我们回到一开始配置的网站，进入 Wizard -> Result, 在 Command line 选择 macOS，复制该命令，在终端中粘贴，开始挖矿了。

查看挖矿状态

我们可以进入这个网站 [https://supportxmr.com/
](https://supportxmr.com/
)，输入之前配置是的钱包地址，就可以看到挖矿的状态，下图是我的。
