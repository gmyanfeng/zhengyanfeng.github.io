Mac 效率工具软件推荐-OpenInTerminal

· 2020-01-10 ·

推荐一款提高效率的软件，特别对于程序员来说，软件是以 Finder 的扩展形式存在的，安装后你并不会在 Applications 文件夹中看到它。

总结来说有三点

  1. 可以快速从**终端（Terminal）**中打开某个目录
  2. 可以快速用编辑器打开某个目录或者文件
  3. 可以复制某个文件或者目录的路径

可以去作者的 Github 下载后按照教程就可以完成配置了
