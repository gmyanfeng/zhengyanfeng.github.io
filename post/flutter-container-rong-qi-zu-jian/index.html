Flutter-Container 容器组件

· 2019-07-23 · # flutter

相当于 HTML 中的 div 标签

Alignment

import 'package:flutter/material.dart';

void main() => runApp(MyApp());

class MyApp extends StatelessWidget {
  // This widget is the root of your application.
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      title: 'Container Demo',
      home:Scaffold(

        appBar: AppBar(
          title: Text('Container Demo'),
        ),

        body: Center(
          child: Container(
            child: new Text(
                'Hello, fat panda',
                style: TextStyle(fontSize: 40),
            ),

            alignment: Alignment.topLeft,
          ),
        ),
      )
    );
  }
}

  • bottomCenter：底部居中对齐
  • bottomLeft：底部左对齐
  • bottomRight：底部右对齐
  • center：居中对齐
  • centerLeft：居中左对齐
  • centerRight：居中右对齐
  • topLeft：顶部左对齐
  • topCenter：顶部居中对齐
  • topRight：顶部右对齐

宽，高和颜色属性

child: Container(
            child: new Text(
                'Hello, fat panda',
                style: TextStyle(fontSize: 40),
            ),

            alignment: Alignment.center,

            width: 500,
            height: 400,
            color: Colors.redAccent,
          ),

padding 内边距

Container 边缘和 child 内容的距离。padding 为 100：

padding: const EdgeInsets.all(100),

上边距：40，左边距为：20

// 从命名 LTRB 可以看出分别时 left,top,right,bottom
padding: const EdgeInsets.fromLTRB(20, 40, 0, 0),

margin

外边距，container 和 外部元素的距离

margin: const EdgeInsets.all(40),

decoration

decoration 是 container 的修饰器，设置背景和边框。
BoxDecoration 给背景加入一个渐变的效果（如果设置了 decoration，就不能设置 color 属性了，这样会产生冲突）

body: Center(
          child: Container(
            child: new Text(
                'Hello, fat panda',
                style: TextStyle(fontSize: 40, backgroundColor: Colors.lightBlue),
            ),

            alignment: Alignment.center,

            width: 500,
            height: 400,
//            padding: const EdgeInsets.all(100),
            padding: const EdgeInsets.fromLTRB(20, 40, 0, 0),
            
            margin: const EdgeInsets.all(40),

            decoration: new BoxDecoration(
              gradient: const LinearGradient(colors: [Colors.lightBlue, Colors.greenAccent, Colors.purple])
            ),
          ),
        ),

记得设置 decoration 要把设置背景颜色的代码去除

border 边框

decoration: new BoxDecoration(
              gradient: const LinearGradient(colors: [Colors.lightBlue, Colors.greenAccent, Colors.purple]),

              border: Border.all(width: 5.0, color: Colors.red)
            ),
